This is a post that contains all of the standard markdown styles supported in ghost. See "A Full Style Test" for a comprehensive test of all HTML elements.

h3: Lists

Unordered list:

  • item f
  • item c
  • item d

Ordered list:

  1. first
  2. second
  3. third

Nested unordered list:

  • unordered item
    • sub-item
    • another sub-item
  • unordered item

Nested ordered list:

  1. first item
    1. first sub-item
    2. second sub-item
  2. second item

Mixed unordered list:

  • unordered item
    1. Ordered sub-item
    2. Ordered sub-item
  • unordered item
  • unordered item

Mixed ordered list:

  1. first item
    • unordered sub-item
    • unordered sub-item
  2. second item

h4: Tables

Ghost doesn't support markdown tables, they're just rendered as text.

h5: Emoji

Ghost doesn't support emoji, that's github you're thinking of. :poop:
